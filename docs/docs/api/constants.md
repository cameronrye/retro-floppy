# Constants

Exported constants and defaults.

## SIZE_MAP

Mapping of size names to pixel values:

```typescript
const SIZE_MAP = {
  tiny: 60,
  small: 120,
  medium: 200,
  large: 400,
  hero: 600,
};
```

## DEFAULT_THEME

Default theme values:

```typescript
const DEFAULT_THEME: FloppyTheme = {
  diskColor: '#2a2a2a',
  slideColor: '#888',
  backgroundColor: '#f5f5f5',
  labelColor: '#fff',
  labelTextColor: '#333',
};
```
