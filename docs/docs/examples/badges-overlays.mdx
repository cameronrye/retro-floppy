# Badges & Overlays

Add badges and custom overlays to your floppy disks.

## Badge Overlay

```jsx live
function BadgeExample() {
  return (
    <FloppyDisk
      size="medium"
      label={{ name: 'New Release', author: 'v2.0' }}
      badge={
        <span style={{
          background: '#ff4444',
          color: 'white',
          padding: '4px 8px',
          borderRadius: '4px',
          fontSize: '12px',
          fontWeight: 'bold',
        }}>
          NEW
        </span>
      }
    />
  );
}
```
