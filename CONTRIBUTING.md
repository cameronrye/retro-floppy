# Contributing

Thanks for your interest in contributing to **Retro Floppy**!

## Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Build the library:

   ```bash
   npm run build
   ```

3. Run the example app:

   ```bash
   cd example
   npm install
   npm run dev
   ```

## Pull Requests

- Open a GitHub issue or discussion before large changes.
- Keep PRs small and focused where possible.
- Include updates to documentation for any user-facing changes.
- Ensure `npm run build` succeeds before submitting.

## Code of Conduct

By participating in this project, you agree to abide by the [Code of Conduct](./CODE_OF_CONDUCT.md).
